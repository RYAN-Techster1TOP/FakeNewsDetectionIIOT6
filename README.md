# FakeNewsDetectionIIOT6
This is our repository for IIOT-6 Project task based on NLP modules used to build a Fake News Detection System. The purpose of this system is to develop am AI powered, web-based Fake News Detection application that uses Natural Language Processing (NLP) models to classify news articles as Real or Fake. The system will allow users to input news text or articles and receive a prediction result based on trained machine learning models.
